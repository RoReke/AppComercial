# AppComercial
La herramienta mapea las empresas de Argentina y brinda información de las mismas con el objetivo de encontrar oportunidades de ventas de seguros de ART. Permite que el usuario guarde nueva información, permite descargar en una tabla la información solicitada. Cuenta con un botón swicht asociado a un modelo de oferta y demanda, que busca filtrar aquellas empresas con precio de mercado considerados como caros. (el modelo no se encuentra en el repositorio por respeto al uso de información privada)<br/>
La app ya cuenta con una muestra del dataframe, por lo tanto NO hay que ejecutar "arma_df.r". 
Para testear la app simplemente abrir el archivo "Server.R" desde Rstudio y darle al botón de "Runapp"
